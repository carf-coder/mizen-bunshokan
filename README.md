# 未然文書館 — The Archive of Might-Have-Beens

**📖 読む: https://carf-coder.github.io/mizen-bunshokan/**

起こらなかった出来事だけを収蔵する架空の公文書館から、AIと人間の協働チームが**毎日1篇**の反実仮想小説を開架する公開実験です。

- **本館**: 日刊ショートショート(その日の現実の断片を「種」に、制約カードを引いて生成)
- **別館『未然欄』**: 国勢調査に「選ばなかった人生を書く欄」があったら——という長編の分冊連載
- **已然図書館『ムレンガ』**: 1964年ザンビア独立の日に生まれた鉱山労働者の一生を、実データに固定して描く調査型伝記小説
- **科学史シリーズ**: 助手の視点・当時の知識だけで科学史を追体験する学習読み物(準備中)

## 特徴: 全工程を開示する

作品はAI(Claude)が執筆し、別のAIインスタンス(検収係)が安全・品質・表象倫理の検査を行い、
PASSしたものだけが自動公開されます。各篇末尾の**目録カード**に種・制約・製作過程を開示。
検収に通らなかった日は「欠番届」が開架されます。この透明性そのものが本プロジェクトの実体です。

## 運営

人間(館長)の委任に基づく自律運転。設計・検収規程・判断記録は非公開リポジトリで管理されています。
